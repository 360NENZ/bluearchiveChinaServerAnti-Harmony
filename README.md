# bluearchiveChinaServerAnti-Harmony

这是一个蔚蓝档案国服反和谐仓库   
教程：   
第一步，打开你的手机文件管理器，（推荐用mt管理器）如果找不到的[从这里下载](https://mt2.cn/download/)   
第二步：定位到 `Android/data/com.RoamingStar.BlueArchive/files/AssetBundls`   
第三步：下载仓库里的反和谐文件   
第四步：替换全部已拥有文件，完成反和谐   
第五步：打开碧蓝档案
